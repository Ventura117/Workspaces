    • Strings

    String(value)

  • Converts a value to a string
  • Returns a new string; does not modify the original value

    • string.split('separator', limit)

  • Splits a string into an array
  • Separator defines where to create a new array value
  • Limit used to specify the number of values to add to the array before stopping the function; is optional
  • Returns a new array; does not modify the original value

    • Example: let name = 'Anthony Ventura' => let nameSeparated = name.split(' ') => nameSeparated = ['Anthony', 'Ventura']

    Example: let names = 'Anthony Ventura Miguel Gutierrez Patrick Fagel' => let namesSeparated = names.split(' ', 3) => namesSeparated = ['Anthony', 'Ventura', 'Miguel']

    array.join('separator')

  • Joins values from an array into a string
  • Separator defines the character to add between values
  • Returns a new string; does not modify the original array

    • let namesSeparated = ['Anthony', 'Ventura'] => let namesJoined = namesSeparated.join(' ') => namesJoined = 'Anthony Ventura'

    string.trim()

  • Removes leading and trailing white space
  • Returns a new string; does not modify the original string

    • string.toUpperCase()

  • Converts all characters in a string to uppercase
  • Returns a new string; does not modify the original string

    • string.toLowerCase()

  • Converts all characters in a string to uppercase
  • Returns a new string; does not modify the original string

    • Numbers

    Number(value)

  • Converts a value to a number
  • Returns a new number; does not modify the original value

    • Math.round(number)

  • Rounds a number to the nearest integer, up or down
  • Returns a new number; does not modify the original value

    • Math.floor(number)

  • Rounds a number down to the nearest integer
  • Returns a new number; does not modify the original value

    • Math.ceil(number)

  • Rounds a number up to the nearest integer
  • Returns a new number; does not modify the original value

    • Math.max(...arr) or Math.max(value1, value2, ...)

  • Find the highest value in a set of values
  • Spread operator must be used when searching an array
  • Returns a new number; does not modify the original values
  • Returns NaN if at least one value cannot be ocnverted to a number

    • Math.min(...arr) or Math.min(value1, value2, ...)

  • Find the lowest value in a set of values
  • Spread operator must be used when searching an array
  • Returns a new number; does not modify the original values
  • Returns NaN if at least one value cannot be ocnverted to a number

    • Objects

    object.property

  • DOT Notation
  • Use when accessing an object where the property name is known
  • Creates a new property under the object if the property name does not exist

    • object[property]

  • Bracket Notation
  • Use when accessing an object where the property name is not known
  • Creates a new property under the object if the property name does not exist

    • Object.keys(objName)

  • Returns an array of an objects property names

    • Example: const person = {firstName: 'Anthony', lastName: 'Ventura', age: 31} => const propertyKeys = Object.keys(person) => propertyKeys = ['firstName', 'lastName', 'age']

    Object.values(objName)

  • Returns an array of an objects property values

    • Example: const person = {firstName: 'Anthony', lastName: 'Ventura', age: 31} => const propertyValues = Object.values(person) => propertyvalues = ['Anthony', 'Ventura', 31]

    Object.entries(objName)

  • Returns an array containing sub arrays of the bojects property-value pairs

    • Example: const person = {firstName: 'Anthony', lastName: 'Ventura', age: 31} => const propertyPairs = Object.entries(person) => propertyPairs = [['firstName', 'Anthony'], ['lastName', 'Ventura'], ['age', 31]]

    Object.assign(target, ...sources)

  • Copies properties and values from one or more objects to the target object
  • Creates shallow copies of nested objects (copies only the reference)
  • Returns a new object; the target object is not modified

    • Arrays

    ...arrayName

  • Expands an array into individual values
  • Not using the spread operator only targets the reference the the array

    • Example: let array1 = [1, 2, 3], let array2 = [4, 5, 6] => arraysCombined = [...array1, ...array2] => arraysCombined = [1, 2, 3, 4, 5, 6]

    arrayName.sort() or arrayName.sort((a, b) => a - b)

  • Sorts values in an array in ascending order
  • Use ((a, b) => a - b) when sorting numbers
  • Modifies the original array

    • Example: let numbers = [3, 1, 2] => numbers.sort() => numbers = [1, 2, 3]

    Example: let letters = ['c', 'a', 'b'] => letters.sort() => letters = ['a', 'b', 'c']

    arrayName.sort((a, b) => b - a)

  • Sorts values in an array in descending order
  • Modifies the original array

    • arrayName.reverse()

  • Reverse the order of values in an array
  • Modifies the original array

    • arrayName.includes(value, startIndex)

  • Search if an arrays contained the given value
  • startIndex defines the index in the array to begin searching; it is optional
  • Returns a boolean

    • arrayName.unshift(value1, value2, ...array, array)

  • Adds values to the beginning of an array
  • Modifies the original array

    • arrayName.shift()

  • Removes the first value from an array
  • Modifies the original array
  • Optionally, returns the value removed

    • arrayName.push(value1, value2, ...array, array)

  • Adds values to the end of an array
  • Modifies the original array

    • arrayName.pop()

  • Removes the last value from an array
  • Modifies the original array
  • Optionally, returns the value removed

    • arrayName.splice(startIndex, deleteCount, value1, value2, ...)

  • Remove or replaces values in an array
  • startIndex defines the position in the array to begin replace/remove; if negative then starts from the end
  • deleteCount defines the number of values to remove remove
  • values define the values to add starting from the startIndex; is optional
  • Optionally, returns an array of the values removed

    • Example: let groceries = ['eggs', 'ham', 'cheese', 'burger'] => groceries.splice(2, 1) => groceries = ['eggs', 'ham', 'burger']

    array.slice(startIndex, endIndex)

  • Returns a shallow copy of array values
  • startIndex defines the the position in the array to begin copying; is optional
  • endIndex defines the position in the array to end copying; is optional
